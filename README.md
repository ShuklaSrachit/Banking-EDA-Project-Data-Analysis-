## Banking EDA Project (Data Analysis)
 Disclaimer: This project is created for educational and analytical purposes only.

The dataset used in this project is synthetic and does not represent real customer data. 
Any resemblance to actual persons, accounts, or financial institutions is purely coincidental.

This project demonstrates exploratory data analysis (EDA) techniques applied to a simulated banking dataset.
